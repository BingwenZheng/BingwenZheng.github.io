MySQL 数据库

启动与停止

命令行窗口输入 :

1.services.msc

2.net start mysql80 (管理员身份运行)

3.net stop mysql80

DDL

  • 查询

    show databases && select database()

  • 创建

    create database

  • 删除

    drop database [if exists]

  • 使用

    use +

对表进行查询操作：

  • 查询所有表

    show tables

  • 查询表结构

    desc + table

  • 查询指定的建表语句

创建表

create table 表名（
字段1 类型 comment ，… ，最后一句不用加逗号）comment

数值类型(可加unsigned) :

  • tinyint 1 byte
  • smallint 2
  • mediumint 3
  • int 4
  • bigint 8
  • float 4
  • double(整体长度，小数长度) 8
  • decimal(依赖精度和标度)

字符串类型 ：

  • char 定长，性能优越
  • verchar 不定长，性能差
  • (tiny medium big)BLOB — 二进制
  • (tiny medium big)TEXT

日期类型

  • date
  • time
  • year
  • datetime
  • timestamp 混合日期与时间，时间戳

SQL

修改表

  • 修改数据类型

    ALTER TABLE 表名 MODIFY 字段名 新数据类型（长度）

  • 修改字段名和类型

    ALTER TABLE CHANGE 旧字段名 新字段名 类型（长度） [注释][约束]

  • 删除字段

    ALTER TABLE 表名 DROP 字段名;

  • 修改表名

    ALTER TABLE 表名 RENAME TO 新表名

  • 删除表名

    DROP TABLE 表名;

DML(增删改查)

  • 增加数据

    给指定字段添加数据 INSERT INTO 表名 （字段1，字段2，…） VALUES (值1，值2，…)
    给全部字段添加数据 INSERT INTO 表名 VALUES （值1，值2，…）
