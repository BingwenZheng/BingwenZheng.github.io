学习资料：b站heima前15p

是什么？

本质是一个项目管理工具，将项目开发和管理过程抽象成一个项目对象模型（POM）

Maven基础概念

仓库 用于存储资源，包含各种jar包

  • 中央仓库
  • 远程仓库

    坐标 ：描述仓库中资源的位置

坐标主要组成：

  • groupid 隶属
  • artifactid 模块名称
  • version
  • mvnrepository.com(导入依赖)

依赖配置

  • 依赖管理：
    层级浅，配置顺序靠后的优先级较高
  • 可选依赖： 将optional属性改为true
  • 排除依赖

    scope属性(作用范围)

  • test 测试代码可用
  • provided 不参与打包
  • runtime 仅支持打包

生命周期与插件

  • clean
  • default（最核心）
  • site

    使用插件可以在生命周期执行不同的功能
